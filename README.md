## ğŸš€ Installation

### Prerequisites
- **Node.js** (v16.0 or higher)
- **npm** or **yarn**
- **Git**

### Quick Start
```bash
# Clone the repository
git clone https://github.com/khush196/SurakshaNET-Dashboard.git

# Navigate to project directory
cd SurakshaNET-Dashboard

# Install dependencies
npm install

npm start
```

